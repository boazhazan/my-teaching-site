2. BindProperty

מוסיפים [BindProperty] מעל property בקוד, והמערכת מחברת אוטומטית בין שדה הטופס ל-property.

Login.cshtml.cs
public class LoginModel : PageModel { [BindProperty] public string Username { get; set; } [BindProperty] public string Password { get; set; } public string Message { get; set; } public void OnPost() { // Username ו-Password כבר מלאים! Message = $"ברוכים הבאים, {Username}!"; } }
Login.cshtml
<form method="post"> <input asp-for="Username" /> <input asp-for="Password" type="password" /> <button type="submit">התחבר</button> </form> @if (Model.Message != null) { <p>@Model.Message</p> }

3. Tag Helpers

Tag Helpers הם תכונות מיוחדות של Razor שמייצרות HTML דינמי:

Tag Helperתפקידדוגמה
asp-forמחבר input ל-Property<input asp-for="Name">
asp-pageקישור לדף Razor<a asp-page="/About">
asp-page-handlerניתוב ל-handler ספציפי<form asp-page-handler="Login">
asp-route-*העברת פרמטרים<a asp-route-id="5">

4. AntiForgeryToken

CSRF (Cross-Site Request Forgery) — התקפה שבה אתר זדוני שולח טופס בשם המשתמש ללא ידיעתו.

ב-Razor Pages, כל <form method="post"> מוסיף אוטומטית שדה נסתר עם token ייחודי. השרת מוודא שה-token תואם — ואם לא, דוחה את הבקשה.

חדשות טובות!

ב-Razor Pages, AntiForgeryToken פועל אוטומטית. לא צריך לעשות כלום — רק לא לשכוח method="post" בטופס.
