שיעור 2.3 — פונקציות

הגדרה, פרמטרים, return, Arrow Functions ו-Scope

1. למה צריך פונקציות?

DRY — Don't Repeat Yourself. פונקציה היא "מתכון" — כותבים פעם אחת, משתמשים שוב ושוב.

// ❌ ללא פונקציה — חוזרים על אותו קוד console.log("שלום דנה!"); console.log("שלום יוסי!"); console.log("שלום מיכל!"); // ✓ עם פונקציה function greet(name) { console.log("שלום " + name + "!"); } greet("דנה"); greet("יוסי"); greet("מיכל");

2. הגדרה וקריאה

פונקציה מוגדרת עם function, שם, סוגריים (עם פרמטרים) וגוף בסוגריים מסולסלים.

function add(a, b) { return a + b; } let result = add(3, 5); console.log(result); // 8
  • פרמטרים — המשתנים שמוגדרים בהגדרת הפונקציה (a, b)
  • ארגומנטים — הערכים שנשלחים בקריאה (3, 5)
function calcArea(width, height) { return width * height; } console.log("שטח: " + calcArea(4, 7)); // 28 console.log("שטח: " + calcArea(10, 3)); // 30

3. ערך חזרה — return

return מחזיר ערך מהפונקציה ועוצר את הריצה — כל קוד אחרי return לא ירוץ.

function isEven(num) { return num % 2 === 0; } console.log(isEven(4)); // true console.log(isEven(7)); // false
פונקציה בלי return

אם לא כתבתם return, הפונקציה מחזירה undefined. זה לא באג — אבל כדאי לדעת.

4. Arrow Functions (פונקציות חץ)

תחביר מקוצר שהתווסף ב-ES6. שימושי במיוחד לפונקציות קצרות:

רגילהחץ
function double(x) { return x * 2; }const double = (x) => x * 2;
function greet(name) { return "Hi " + name; }const greet = (name) => "Hi " + name;
const double = (x) => x * 2; const sum = (a, b) => a + b; const hello = () => "שלום עולם!"; console.log(double(5)); // 10 console.log(sum(3, 4)); // 7 console.log(hello()); // שלום עולם!
מתי להשתמש בחץ?

פונקציות חץ מצוינות לפונקציות קצרות ול-callbacks. לפונקציות ארוכות עם לוגיקה מורכבת, עדיין מקובל להשתמש ב-function הרגיל.

5. Scope — תחום הכרה

משתנה "מוכר" רק בתוך הבלוק שבו הוגדר.

let globalVar = "אני גלובלי"; function test() { let localVar = "אני מקומי"; console.log(globalVar); // ✓ עובד console.log(localVar); // ✓ עובד } test(); console.log(globalVar); // ✓ עובד // console.log(localVar); // ❌ ReferenceError!
