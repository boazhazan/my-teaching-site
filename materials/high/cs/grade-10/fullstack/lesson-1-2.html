עיצוב ב-CSS ומודל הקופסה

סלקטורים, Box Model, צבעים, פונטים ותכונות נפוצות

1. מה זה CSS?

CSS הוא קיצור של Cascading Style Sheets — שפת עיצוב שמגדירה איך אלמנטי HTML נראים על המסך.

בעוד HTML מגדיר את המבנה של הדף (כותרות, פסקאות, תמונות), CSS אחראי על המראה: צבעים, גדלים, מרווחים, פונטים, פריסה ועוד.

בלי CSS, כל דפי האינטרנט היו נראים כמו מסמכי טקסט פשוטים — בלי צבע, בלי עיצוב, בלי אישיות.

💡 אנלוגיה שימושית

HTML = השלד של הבניין, CSS = העיצוב והצבע של הקירות, JavaScript = החשמל והמעלית (ההתנהגות).

2. שלוש שיטות לכתיבת CSS

יש שלוש דרכים להוסיף CSS לדף HTML. לכל שיטה יתרונות וחסרונות:

Inline

ישירות על התגית

Internal

בתוך תגית <style>

External ✓

קובץ .css נפרד

שיטה 1: Inline CSS

כותבים את העיצוב ישירות בתוך מאפיין style של התגית:

<p style="color: red; font-size: 20px;">טקסט אדום</p>

שיטה 2: Internal CSS

כותבים בלוק CSS בתוך תגית <style> בחלק ה-<head> של הדף:

<head> <style> p { color: blue; font-size: 18px; } </style> </head>

שיטה 3: External CSS (מומלצת)

כותבים את ה-CSS בקובץ נפרד (למשל style.css) ומקשרים אותו עם תגית <link>:

<!-- בקובץ HTML --> <head> <link rel="stylesheet" href="style.css"> </head> /* בקובץ style.css */ p { color: green; font-size: 16px; }
⚠️ שימו לב

Inline CSS קשה לתחזוקה — כי העיצוב מפוזר בכל התגיות. עדיף להשתמש ב-Internal (לדפים בודדים) או External (לפרויקטים אמיתיים) כדי לשמור על קוד נקי ומסודר.

3. סלקטורים (Selectors)

סלקטורים הם הדרך של CSS "לבחור" את האלמנטים שרוצים לעצב. שלושת הסלקטורים הבסיסיים:

סלקטור תגית (Element Selector)

בוחר את כל האלמנטים מאותו סוג:

p { color: blue; } /* כל הפסקאות בדף יהיו בצבע כחול */

סלקטור מזהה (ID Selector)

בוחר אלמנט ספציפי אחד לפי ה-ID שלו. מסומן ב-#:

<!-- HTML --> <h1 id="title">כותרת ראשית</h1> /* CSS */ #title { font-size: 32px; color: darkblue; }

סלקטור מחלקה (Class Selector)

בוחר קבוצה של אלמנטים שחולקים את אותה מחלקה. מסומן ב-.:

<!-- HTML --> <p class="highlight">טקסט מודגש</p> <p>טקסט רגיל</p> <p class="highlight">גם זה מודגש</p> /* CSS */ .highlight { background-color: yellow; font-weight: bold; }
✅ כלל אצבע

ID — ייחודי, אלמנט אחד בלבד (כמו תעודת זהות). Class — משותף, הרבה אלמנטים יכולים לשתף אותה מחלקה (כמו חולצת קבוצה).

4. מודל הקופסה (Box Model)

כל אלמנט HTML הוא בעצם קופסה מלבנית. מודל הקופסה מגדיר את ארבע השכבות של כל אלמנט:

margin
border
padding
Content
התוכן
  • Content — התוכן עצמו: טקסט, תמונה וכו'
  • Padding — ריפוד: המרווח בין התוכן לגבול (border). שקוף, אבל "דוחף" את התוכן פנימה
  • Border — גבול: קו שמקיף את האלמנט. יכול להיות בצבע, עובי וסגנון שונים
  • Margin — שוליים: המרווח מחוץ לגבול, בין האלמנט לשכנים שלו
.card { width: 300px; /* רוחב התוכן */ padding: 20px; /* ריפוד פנימי */ border: 2px solid #333; /* גבול: עובי, סגנון, צבע */ margin: 16px auto; /* שוליים: למעלה/למטה 16px, מרכוז אופקי */ }
💡 טיפ חשוב: box-sizing

כברירת מחדל, width מגדיר רק את רוחב התוכן — ה-padding וה-border נוספים מעליו. כדי שה-width יכלול הכל, הוסיפו: box-sizing: border-box;

5. תכונות CSS נפוצות

הנה טבלת עזר מהיר עם התכונות הנפוצות ביותר:

תכונה תפקיד דוגמה
color צבע הטקסט color: #2563eb;
background-color צבע רקע background-color: #f0f0f0;
font-size גודל גופן font-size: 18px;
font-family סוג גופן font-family: Arial, sans-serif;
text-align יישור טקסט text-align: center;
width / height רוחב / גובה width: 300px;
border-radius עיגול פינות border-radius: 12px;
padding ריפוד פנימי padding: 16px;
margin שוליים חיצוניים margin: 20px auto;

📝 תרגילים
