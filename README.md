# my-teaching-site

## Structure

```
index.html        — Home page (hero, features, levels, testimonials)
about.html        — About the tutor
services.html     — Lesson types & subjects
materials.html    — Learning materials hub (tabbed by level)
contact.html      — Contact form & info
css/style.css     — Main stylesheet (RTL, responsive)
js/main.js        — Navigation, tabs, form handling
assets/images/    — Image assets (placeholder)
```
