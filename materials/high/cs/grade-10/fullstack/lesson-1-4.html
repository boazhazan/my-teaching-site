יחידה 1: Frontend — HTML & CSS

שיעור 1.4 — טבלאות, רשימות ו-Bootstrap

נלמד להציג נתונים בטבלאות וברשימות, ואז נכיר את Bootstrap — ספריית CSS שחוסכת עבודה רבה

1. טבלאות HTML

טבלה היא אחת הדרכים הנפוצות ביותר להציג מידע מסודר באינטרנט — ציונים, לוחות זמנים, מחירונים ועוד. ב-HTML יוצרים טבלה באמצעות ארבע תגיות עיקריות:

  • <table> — עוטפת את כל הטבלה
  • <tr> — שורה (Table Row)
  • <th> — תא כותרת (Table Header) — מודגש וממורכז אוטומטית
  • <td> — תא רגיל (Table Data)

בואו ניצור מערכת שעות — דוגמה קלאסית לשימוש בטבלה:

<table border="1"> <tr> <th>שעה</th> <th>ראשון</th> <th>שני</th> <th>שלישי</th> </tr> <tr> <td>08:00</td> <td>מתמטיקה</td> <td>אנגלית</td> <td>מדעי המחשב</td> </tr> <tr> <td>09:00</td> <td>עברית</td> <td>היסטוריה</td> <td>פיזיקה</td> </tr> <tr> <td>10:00</td> <td>ספורט</td> <td>מתמטיקה</td> <td>אנגלית</td> </tr> </table>
שימו לב

המאפיין border="1" מוסיף גבול בסיסי לטבלה. בפרקטיקה, נשתמש ב-CSS (או ב-Bootstrap) כדי לעצב טבלאות בצורה יפה יותר.

תכונות שימושיות

  • colspan="2" — תא שמתפרס על 2 עמודות
  • rowspan="2" — תא שמתפרס על 2 שורות
  • <thead>, <tbody>, <tfoot> — חלוקה סמנטית של הטבלה לראש, גוף ותחתית

2. רשימות

רשימות מאפשרות להציג פריטים בצורה מסודרת. יש שני סוגים עיקריים:

רשימה ממוספרת (Ordered List)

כשלסדר יש משמעות — שלבים בתהליך, דירוג, מתכון:

<h3>איך ליצור אתר?</h3> <ol> <li>כתוב את ה-HTML</li> <li>הוסף עיצוב עם CSS</li> <li>הוסף אינטראקציה עם JavaScript</li> <li>העלה לשרת</li> </ol>

רשימה לא ממוספרת (Unordered List)

כשהסדר לא חשוב — רשימת קניות, תכונות של מוצר:

<h3>מה צריך למפתח Full-Stack?</h3> <ul> <li>HTML & CSS</li> <li>JavaScript</li> <li>שפת צד שרת (C#, Python...)</li> <li>בסיסי נתונים (SQL)</li> </ul>
טיפ

אפשר לשלב רשימות בתוך רשימות (Nested Lists) — פשוט שמים <ul> או <ol> בתוך <li>. זה שימושי לתפריטים ולמבנים היררכיים.

3. מה זה Bootstrap?

Bootstrap היא ספריית CSS (ו-JavaScript) פופולרית שנוצרה במקור על ידי צוות Twitter. היא מספקת עיצוב מוכן מראש — כפתורים, טפסים, כרטיסים, ניווט ועוד — כך שלא צריך לכתוב CSS מאפס.

למה להשתמש ב-Bootstrap?

  • חוסך זמן — עיצוב מקצועי בלי לכתוב מאות שורות CSS
  • Responsive — האתר נראה טוב גם בנייד, גם במחשב
  • עקבי — כל הרכיבים תואמים אחד לשני בסגנון
  • תיעוד מצוין — getbootstrap.com מסביר כל רכיב עם דוגמאות

איך מוסיפים Bootstrap לפרויקט?

הדרך הפשוטה ביותר — קישור CDN בתוך <head>:

<head> <!-- Bootstrap 5 CSS --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <!-- Bootstrap 5 JS (לפני סגירת body) --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
חשוב

CDN = Content Delivery Network. זה אומר שהקובץ נטען משרת חיצוני. צריך חיבור אינטרנט כדי שזה יעבוד. בפרויקט אמיתי, אפשר גם להוריד את הקובץ ולשמור מקומית.

מרגע שהוספנו את הקישור, אפשר להשתמש ב-class מוכנים של Bootstrap. למשל:

<button class="btn btn-primary">לחץ כאן</button> <button class="btn btn-success">שמור</button> <button class="btn btn-danger">מחק</button>

4. מערכת Grid

אחד הכלים החזקים ביותר ב-Bootstrap הוא מערכת ה-Grid — מערכת של 12 עמודות שמאפשרת לסדר תוכן בשורות ועמודות בקלות.

המבנה הבסיסי

  • container — עוטף הכל, מרכז את התוכן ומגביל רוחב
  • row — שורה אופקית
  • col — עמודה בתוך השורה

כל שורה מחולקת ל-12 חלקים. אם רוצים שני חצאים, משתמשים ב-col-6 + col-6. אם רוצים שליש ושני-שלישים: col-4 + col-8.

<div class="container"> <div class="row"> <div class="col-md-6" style="background:#dbeafe;padding:20px"> חצי שמאל (6 מתוך 12) </div> <div class="col-md-6" style="background:#fef3c7;padding:20px"> חצי ימין (6 מתוך 12) </div> </div> <div class="row mt-3"> <div class="col-md-4" style="background:#d1fae5;padding:20px"> שליש (4 מתוך 12) </div> <div class="col-md-4" style="background:#e0e7ff;padding:20px"> שליש (4 מתוך 12) </div> <div class="col-md-4" style="background:#fce7f3;padding:20px"> שליש (4 מתוך 12) </div> </div> </div>
Responsive Breakpoints

הקידומת md פירושה "מסך בינוני" (768px+). במסכים קטנים יותר, העמודות ייערמו אחת מתחת לשנייה. קידומות נוספות: sm (קטן), lg (גדול), xl (גדול מאוד).

5. רכיבי Bootstrap

Bootstrap מגיעה עם עשרות רכיבים מוכנים. נכיר שניים שימושיים במיוחד:

כרטיסים (Cards)

כרטיס הוא מסגרת מעוצבת להצגת תוכן — תמונה, כותרת, טקסט וכפתור:

<div class="card" style="width:18rem"> <img src="https://via.placeholder.com/300x200" class="card-img-top" alt="תמונה"> <div class="card-body"> <h5 class="card-title">כותרת הכרטיס</h5> <p class="card-text">טקסט תיאורי קצר על התוכן.</p> <a href="#" class="btn btn-primary">קרא עוד</a> </div> </div>

ניווט (Navbar)

סרגל ניווט רספונסיבי — במחשב מוצג כתפריט אופקי, בנייד הופך לכפתור המבורגר:

<nav class="navbar navbar-expand-lg navbar-dark bg-dark"> <div class="container-fluid"> <a class="navbar-brand" href="#">האתר שלי</a> <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navMenu"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="navMenu"> <ul class="navbar-nav me-auto"> <li class="nav-item"> <a class="nav-link active" href="#">דף הבית</a> </li> <li class="nav-item"> <a class="nav-link" href="#">אודות</a> </li> <li class="nav-item"> <a class="nav-link" href="#">צור קשר</a> </li> </ul> </div> </div> </nav>
עוד רכיבים שכדאי להכיר

Bootstrap כוללת גם Alerts (הודעות), Modals (חלונות קופצים), Carousel (מצגת תמונות), Tables (טבלאות מעוצבות עם class="table") ועוד. אפשר לחפש הכל בתיעוד הרשמי: getbootstrap.com/docs

שימו לב שהוספנו class="table" — זה כל מה שצריך כדי להפוך טבלת HTML רגילה לטבלה מעוצבת ב-Bootstrap:

<table class="table table-striped table-hover"> <thead> <tr> <th>שם</th> <th>ציון</th> <th>סטטוס</th> </tr> </thead> <tbody> <tr> <td>יוסי</td> <td>95</td> <td><span class="badge bg-success">עבר</span></td> </tr> </tbody> </table>

עורך קוד חי

כתבו קוד HTML עם Bootstrap ולחצו "הרץ" כדי לראות את התוצאה:

תרגיל

צרו דף Landing Page בנושא שאתם אוהבים — משחק, סרט, ענף ספורט, להקה, או כל דבר אחר. הדף צריך לכלול:

  1. Navbar — סרגל ניווט עם לפחות 3 קישורים
  2. כרטיסים (Cards) — לפחות 3 כרטיסים עם כותרת, טקסט וכפתור
  3. טבלה — עם לפחות 3 שורות (למשל: דירוג, סטטיסטיקות, לוח זמנים)
  4. רשימות — רשימה ממוספרת אחת ורשימה לא ממוספרת אחת
  5. מערכת Grid — שימוש ב-container, row ו-col לפריסת התוכן

השתמשו בעורך הקוד למעלה — ערכו את הקוד לפי הנושא שבחרתם ולחצו "הרץ" כדי לבדוק.
