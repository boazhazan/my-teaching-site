שיעור 2.4 — אירועים ועבודה עם DOM

onclick, getElementById, innerHTML, addEventListener

1. מה הם אירועים?

אירוע הוא כל דבר שקורה בדף — לחיצה, הקלדה, תזוזת עכבר. ב-JavaScript אנחנו "מקשיבים" לאירועים ומגיבים.

אירועמתי קורה
onclickלחיצה על אלמנט
ondblclickלחיצה כפולה
onchangeשינוי ערך (select, input)
onmouseoverעכבר נכנס לאלמנט
onmouseoutעכבר יוצא מאלמנט
onfocusשדה קלט מקבל פוקוס
onblurשדה קלט מאבד פוקוס
<!-- אירוע inline --> <button onclick="alert('לחצת!')">לחץ עליי</button>

2. אירועי מקלדת

שלושה אירועים: onkeydown (מקש נלחץ), onkeyup (מקש שוחרר), onkeypress (תו הוקלד).

<input type="text" id="myInput" onkeydown="showKey(event)"> <p id="output"></p> <script> function showKey(event) { document.getElementById("output").innerHTML = "מקש: " + event.key + " | קוד: " + event.keyCode; } </script>

3. עבודה עם DOM

ה-DOM (Document Object Model) הוא ייצוג של דף ה-HTML כאובייקטים. עם JS אנחנו יכולים לגשת לכל אלמנט ולשנות אותו.

שיטות גישה

// גישה לפי ID let el = document.getElementById("myDiv"); // שינוי תוכן HTML el.innerHTML = "<b>שלום!</b>"; // שינוי טקסט בלבד (בטוח יותר) el.textContent = "שלום!"; // קריאת/שינוי ערך של input let val = document.getElementById("nameInput").value; document.getElementById("nameInput").value = "ערך חדש";
innerHTML לעומת textContent

innerHTML מפרש HTML (יכול ליצור אלמנטים חדשים). textContent מציג טקסט גולמי. לטקסט רגיל — עדיף textContent.

4. שינוי עיצוב דרך JS

let box = document.getElementById("box"); // שינוי ישיר box.style.color = "red"; box.style.backgroundColor = "#dbeafe"; box.style.fontSize = "20px"; box.style.padding = "12px"; // שימוש ב-classList box.classList.add("active"); box.classList.remove("hidden"); box.classList.toggle("dark-mode"); // מוסיף/מסיר
classList עדיף!

שינוי ישיר ב-style מוסיף inline CSS. עדיף להגדיר class ב-CSS ולהשתמש ב-classList.toggle() לניקיון.

5. addEventListener — הגישה המודרנית

במקום onclick ב-HTML, הגישה המומלצת היא addEventListener ב-JS:

let btn = document.getElementById("myBtn"); btn.addEventListener("click", function() { alert("לחצת!"); }); // אפשר לחבר כמה מאזינים לאותו אלמנט! btn.addEventListener("click", function() { console.log("גם אני רץ!"); });

סוגי אירועים: 'click', 'input', 'change', 'keydown', 'mouseover', 'submit'
