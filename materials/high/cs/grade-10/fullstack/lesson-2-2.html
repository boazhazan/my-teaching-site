if / else, switch, for, while

1. תנאי if / else if / else

תנאים מאפשרים לתוכנית לקבל החלטות. הקוד בתוך הבלוק ירוץ רק אם התנאי מתקיים.

let grade = 85; if (grade >= 90) { console.log("מצוין!"); } else if (grade >= 70) { console.log("טוב מאוד"); } else if (grade >= 55) { console.log("עובר"); } else { console.log("נכשל"); }
שימו לב לסדר!

התנאים נבדקים מלמעלה למטה. ברגע שתנאי מתקיים, שאר הבלוקים מדולגים. לכן שימו את התנאי הכי מחמיר ראשון.

3. switch-case

כשיש הרבה ערכים אפשריים לבדוק, switch קריא יותר מ-if מרובה:

let day = new Date().getDay(); switch (day) { case 0: console.log("יום ראשון"); break; case 1: console.log("יום שני"); break; case 2: console.log("יום שלישי"); break; case 3: console.log("יום רביעי"); break; case 4: console.log("יום חמישי"); break; case 5: console.log("יום שישי"); break; case 6: console.log("שבת"); break; default: console.log("לא ידוע"); }
אל תשכחו break!

בלי break הקוד ימשיך לרוץ גם ב-case הבא (fall-through). זה כמעט תמיד באג.

4. לולאת for

לולאה חוזרת על קוד מספר פעמים ידוע מראש.

// לוח כפל של 7 for (let i = 1; i <= 10; i++) { console.log("7 × " + i + " = " + (7 * i)); }

שלושת חלקי ה-for: אתחול (let i = 1) → תנאי (i <= 10) → קידום (i++).

// סכום מספרים מ-1 עד 100 let sum = 0; for (let i = 1; i <= 100; i++) { sum += i; } console.log("סכום: " + sum); // 5050

5. לולאת while

לולאה שחוזרת כל עוד תנאי מתקיים — שימושית כשלא יודעים מראש כמה פעמים לחזור.

// הכפלה עד שעוברים 1000 let num = 1; let count = 0; while (num < 1000) { num *= 2; count++; } console.log("הגענו ל-" + num + " אחרי " + count + " הכפלות");

break ו-continue

// break — עוצר את הלולאה לגמרי for (let i = 1; i <= 100; i++) { if (i === 50) break; // עוצר ב-50 console.log(i); } // continue — מדלג על הסיבוב הנוכחי for (let i = 1; i <= 10; i++) { if (i % 2 === 0) continue; // מדלג על זוגיים console.log(i); // מדפיס רק אי-זוגיים }
