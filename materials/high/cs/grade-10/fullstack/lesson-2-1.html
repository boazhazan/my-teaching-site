שיעור 2.1 — יסודות השפה ומשתנים

משתנים, סוגי נתונים, קלט/פלט, אופרטורים וגישה ל-DOM

1. מה זה JavaScript?

JavaScript (בקיצור JS) היא שפת תסריט (Scripting Language) שרצה בתוך הדפדפן. בעוד ש-HTML מגדיר את המבנה של הדף ו-CSS מגדיר את העיצוב, JavaScript אחראית על ההתנהגות — אינטראקטיביות, תגובה ללחיצות, שינוי תוכן בזמן אמת ועוד.

שלוש השכבות של דף אינטרנט

  • HTML — מבנה (שלד הדף, תוכן, כותרות, טבלאות)
  • CSS — עיצוב (צבעים, גדלים, מיקומים, אנימציות)
  • JavaScript — התנהגות (לוגיקה, אירועים, מניפולציה של DOM)
💡 ידעת?

JavaScript היא אחת השפות הפופולריות בעולם — היא רצה בצד הלקוח (דפדפן) וגם בצד השרת באמצעות Node.js. כל דפדפן מודרני מכיל מנוע JavaScript מובנה (V8 ב-Chrome, SpiderMonkey ב-Firefox).

איך מוסיפים JavaScript לדף HTML?

משתמשים בתגית <script> — בדרך כלל לפני סגירת </body>:

<body> <h1>הדף שלי</h1> <script> // כאן כותבים קוד JavaScript alert("שלום עולם!"); </script> </body>

אפשר גם לקשר קובץ JS חיצוני: <script src="app.js"></script> — בקורס הזה נכתוב הכל inline בתוך תגית <script>.

2. משתנים (Variables)

משתנה הוא "קופסה" בזיכרון שמחזיקה ערך. ב-JavaScript יש שלוש דרכים להכריז על משתנה:

מילת מפתח תיאור Scope המלצה
var הכרזה ישנה (ES5) Function scope ❌ להימנע
let הכרזה מודרנית (ES6) Block scope ✅ מומלץ
const ערך קבוע שלא משתנה Block scope ✅ מומלץ לקבועים
// var — הכרזה ישנה, פחות בטוחה var name = "בועז"; // let — הכרזה מודרנית, מומלץ! let age = 25; age = 26; // מותר לשנות ערך // const — ערך קבוע const PI = 3.14159; // PI = 3; ← שגיאה! לא ניתן לשנות const

סוגי נתונים (Data Types)

JavaScript היא dynamically typed — לא צריך להכריז על סוג המשתנה, הוא נקבע אוטומטית:

// Number — מספר שלם או עשרוני let score = 100; let temperature = 36.6; // String — מחרוזת (טקסט) let greeting = "שלום עולם"; let city = 'ירושלים'; // Boolean — ערך אמת/שקר let isLoggedIn = true; let isAdmin = false; // בדיקת סוג: console.log(typeof score); // "number" console.log(typeof greeting); // "string" console.log(typeof isLoggedIn); // "boolean"
⚠️ var vs let — מה ההבדל?

var מוגדר ברמת הפונקציה (function scope) — אם תגדירו var בתוך if, הוא יהיה נגיש גם מחוץ ל-if. let מוגדר ברמת הבלוק (block scope) — נגיש רק בתוך הבלוק { } שבו הוכרז. לכן let בטוח יותר ומומלץ.

שרשור מחרוזות (String Concatenation)

let firstName = "בועז"; let lastName = "חזן"; // שרשור עם + let fullName = firstName + " " + lastName; console.log(fullName); // "בועז חזן" // Template Literals (ES6) — מומלץ! let message = `שלום ${firstName}, ברוך הבא!`; console.log(message); // "שלום בועז, ברוך הבא!"

3. קלט / פלט (Input & Output)

JavaScript מספקת מספר דרכים לתקשר עם המשתמש:

פלט — הצגת מידע

// alert — חלונית קופצת alert("שלום! זו הודעה חשובה"); // console.log — הדפסה לקונסול (לפיתוח) console.log("ערך x הוא:", 42); // document.write — כתיבה ישירה לדף document.write("<h2>כותרת מ-JS</h2>");

קלט — קבלת מידע מהמשתמש

// prompt — חלונית עם שדה קלט let name = prompt("מה השם שלך?"); alert("שלום " + name + "!"); // prompt מחזיר מחרוזת — צריך להמיר למספר: let ageStr = prompt("בן כמה אתה?"); let age = parseInt(ageStr); // מחרוזת → מספר שלם let price = parseFloat("19.99"); // מחרוזת → מספר עשרוני // דרך מקוצרת: Number() let num = Number(prompt("הכנס מספר:"));
✅ טיפ חשוב

prompt() תמיד מחזיר מחרוזת (string), גם אם המשתמש הקליד מספר. אם רוצים לעשות חישובים — חובה להמיר עם parseInt(), parseFloat() או Number(). בלי המרה, "5" + "3" ייתן "53" (שרשור!) ולא 8.

אופרטורי השוואה

אופרטורמשמעותדוגמהתוצאה

⚠️ == מול === — הבדל קריטי!

== (שוויון רופף) מבצע המרת סוג אוטומטית (Type Coercion) לפני ההשוואה, לכן 5 == "5" מחזיר true. === (שוויון מוחלט) משווה גם ערך וגם סוג — 5 === "5" מחזיר false.
תמיד השתמשו ב-=== כדי למנוע באגים מפתיעים!

אופרטורים לוגיים

אופרטורשםדוגמהמשמעות
&&AND (וגם)true && falseשניהם חייבים להיות true
||OR (או)true || falseלפחות אחד true
!NOT (שלילה)!trueהופך את הערך
let age = 25; let hasLicense = true; // AND — שני התנאים חייבים להתקיים console.log(age >= 17 && hasLicense); // true // OR — לפחות תנאי אחד console.log(age < 10 || age > 60); // false // NOT — הפיכה console.log(!hasLicense); // false

5. DOM — גישה לאלמנטים בדף

DOM (Document Object Model) הוא ייצוג עץ של דף ה-HTML בזיכרון. JavaScript יכולה לגשת לכל אלמנט בדף, לשנות את תוכנו, עיצובו ואף להוסיף אלמנטים חדשים.

גישה לאלמנט לפי ID

// בחירת אלמנט לפי id let title = document.getElementById("myTitle"); // שינוי תוכן title.innerHTML = "כותרת חדשה!"; // שינוי עיצוב title.style.color = "blue"; title.style.fontSize = "2rem"; title.style.backgroundColor = "#dbeafe";

דוגמה: כפתור שמשנה תוכן

<h1 id="title">שלום!</h1> <button onclick="changeTitle()">לחץ עליי</button> <script> function changeTitle() { let el = document.getElementById("title"); el.innerHTML = "הכפתור נלחץ!"; el.style.color = "green"; } </script>
💡 getElementById vs querySelector

getElementById("id") בוחר לפי id בלבד. querySelector(".class") ו-querySelector("#id") יכולים לבחור לפי כל סלקטור CSS. בשלב הזה נשתמש ב-getElementById — פשוט ויעיל.
